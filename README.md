## Implementación de IntTree, utilizando métodos de ordenamiento: postOrden,intOrden,preOrden.

**Descripcion:**

La clase Node<T> representa el funcionamiento del árbol binario. Cada nodo contiene un valor y dos referencias a sus hijos (izquierdo y derecho), además dependiendo del tipo de ordenamiento a usar. Gracias a esta estructura, es posible construir árboles binarios y aplicar recorridos como PreOrden, InOrden y PostOrden para procesar los elementos almacenados.



## Implementación de BinaryTree, utilizando métodos de ordenamiento: postOrden,intOrdne,preOrden.

**Descripcion:**

La clase BinaryTree nos permite guardar datos en un árbol binario usando nodos enlazados. Incluye los recorridos PreOrden, InOrden y PostOrden para visitar los nodos en diferentes órdenes. También permite insertar elementos y obtener información del árbol como su altura y la cantidad de nodos.
